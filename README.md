# External Attack Surface Intelligence using OSINT

This project demonstrates how Open Source Intelligence (OSINT) techniques can be used to discover and analyze the external attack surface of an organization.

## Tools Used

Amass
theHarvester
Nmap
Shodan
VirusTotal
AbuseIPDB

## Methodology

1. Subdomain discovery using Amass
2. Email intelligence gathering using theHarvester
3. Service enumeration using Nmap
4. Threat intelligence enrichment using VirusTotal and AbuseIPDB
